使い方
---------------------------------------------------------
下記の各iniファイルにコピーしたい内容を記述
Item01.ini
Item02.ini
Item03.ini
Item04.ini
Item05.ini

※iniファイル名は変更しないで下さい
---------------------------------------------------------
書き方は、下記のように = の右側に記述するだけです
[0]
NameID=ボタン名ID
NamePW=ボタン名PW
CopyID=ボタン押下時にコピーする内容ID
CopyPW=ボタン押下時にコピーする内容PW

※ = の左側は変更しないで下さい
---------------------------------------------------------
タブ名は同じiniファイルに記述すれば反映されます
[TabPage]
Tab=通販

※ = の左側は変更しないで下さい
---------------------------------------------------------
